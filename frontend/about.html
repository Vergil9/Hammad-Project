About CampusConnect

Project Objective

CampusConnect was developed as a final semester project for OSSD-Y9 at the University of Management and Technology.

The primary objective is to replace outdated paper-based and legacy digital systems with a unified, high-performance web application. It aims to streamline administrative tasks by providing a centralized hub for managing student records, course enrollments, and user authentication securely.

Target Audience

  • University Administrators: Can oversee total enrollments, monitor system health, and manage faculty and student accounts.
  • Faculty: Can update student grades, manage course materials, and view student lists.
  • Students: Can view their enrolled courses, update profile information, and track their academic progress.

Technology Stack

Frontend

Vanilla HTML5, CSS3 with custom variables, and ES6 JavaScript.

Backend

FastAPI (Python) for ultra-fast, async API endpoints.

Database

Supabase PostgreSQL managed via SQLAlchemy ORM.

Security

JWT token authentication and passlib bcrypt hashing.
